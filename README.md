# dotfiles

setup MacPC

# procedure
homedirectory
1. install [Homebrew](https://brew.sh/index_ja)
2. `mkdir -p ~/.config/brewfile`
3. `cd ~/.config/brewfile` `touch Brewfile`
4. XCODEをインストールしてBrewfile作成
5. `brew install rcmdnk/file/brew-file`
6. `brew file install`
